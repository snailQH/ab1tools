# Visualize Chromatograms

ab1tools provides scriptable, command-line chromatogram visualization — no GUI application required. Generated PNGs are publication-quality and suitable for journal figures.

---

## `plot-ab1` — Export chromatogram PNG from existing AB1

Read any AB1 file (generated by ab1tools or from a real Sanger sequencer) and export a multi-panel chromatogram PNG for any region of interest.

```bash
# Full sequence
ab1tools plot-ab1 sample.ab1 -o full_chromatogram.png

# Specific region (bases 100-200)
ab1tools plot-ab1 sample.ab1 --start 100 --end 200 -o region.png

# Compact layout (200 bases per row)
ab1tools plot-ab1 sample.ab1 --bases-per-row 200 -o compact.png

# High-resolution for publication
ab1tools plot-ab1 sample.ab1 --start 500 --end 600 --bases-per-row 50 --dpi 300 -o figure.png
```

### When to use `plot-ab1`

- **Publication figures**: export specific regions at high DPI with custom layout
- **Variant inspection**: zoom into a SNP or heterozygous site
- **Batch export**: script chromatogram generation for many AB1 files
- **Re-plot existing Sanger data**: read real AB1 files from ABI sequencers

### Parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| `--start` | 1 | Start base position (1-based, inclusive) |
| `--end` | end | End base position (1-based, inclusive) |
| `--bases-per-row` | 100 | Bases per row in the PNG |
| `--dpi` | 150 | PNG resolution (use 300 for publication) |

### Output layout

- Multi-panel: 4 rows per page, configurable bases per row
- Each row shows all 4 channels (A=green, C=blue, G=black, T=red)
- Base letters annotated above peaks
- Position numbers every 10 bases
- Automatic multi-page output for long sequences (page 2 = `_p2.png`)

## `smart` — Automated variant detection + targeted visualization

Smart mode combines AB1 generation with heterozygous site detection and **targeted chromatogram PNGs** that focus only on variant regions.

```bash
ab1tools smart /path/to/Analysis/ -o output/ --noise 5 --phasing 0.1 --decay 0.5
```

### What smart mode generates

1. **AB1 files** for all samples
2. **`heterozygous_sites.csv`** — table of all detected variant positions with allele frequencies
3. **`*_hetero.png`** — targeted chromatogram PNGs showing only the flanking regions around variant sites, with **orange highlighting** at variant positions
4. Samples without variants get AB1 only — no unnecessary PNGs

### Smart mode parameters

| Parameter | Default | Description |
|-----------|---------|-------------|
| `--min-minor-freq` | 0.05 | Minimum minor allele frequency to call heterozygous |
| `--flanking` | 20 | Bases shown on each side of variant in PNG |
| `--no-plot` | off | Skip PNG, generate AB1 + CSV only |

### Example output

```
output/
├── barcode01.ab1               # AB1 chromatogram
├── barcode01_hetero.png        # Targeted PNG (6 variant regions)
├── barcode02.ab1
├── barcode03.ab1
├── barcode03_hetero.png        # Targeted PNG (4 variant regions)
└── heterozygous_sites.csv      # Summary table
```

### Use cases

- **High-throughput screening**: process entire amplicon runs, only review samples with variants
- **CRISPR editing QC**: quickly identify which samples show editing and at what efficiency
- **Variant surveillance**: monitor mutation frequencies across samples over time
